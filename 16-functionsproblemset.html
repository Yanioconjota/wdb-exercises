Functions problem set

Do the following:

  • 1 - Write a function isEven() wich takes a single numeric argument and returns true if the number is even
  • 2 - Write a function factorial() wich takes a single numeric argument and returns the factorial of that number. Ex. 4! is 4 x 3 x 2 x 1. 6! is 6 x 5 x 4 x 3 x 2 x 1. 0! is 1.
  • 3 - Write a function kebabToSnake() which takes a single kebab-cased string and returns the snake_case version.

1 - Write a function isEven() wich takes a single numeric argument and returns true if the number is even

Code example:

          function isEven(num){
            if(num % 2 === 0){
              return true;
            }
          };

But this can be easily refactored by doing this instead:

          function isEven(num){
            return num % 2 === 0;
          };

2 - Write a function factorial() wich takes a single numeric argument and returns the factorial of that number. Ex. 4! is 4 x 3 x 2 x 1. 6! is 6 x 5 x 4 x 3 x 2 x 1. 0! is 1.

Code example:

It can be done in several ways. The first one is by declaring the num variable and setting it to 1 and increasing its value n times until it reaches num's value

          function factorial(num){
            //define a result variable
            var result = 1;
            //calculate factorial and store value in result
            for(var i = 2; i <= num; i++){
              result *= i;
            }
            //return the result variable
            return result;
          };

Also it can be done the other way around, by setting result to num's value and keep decreasing until it goes to 1, the only problem here is when num is equal to 0, the result will be 0, but that can be avoided with an if statement.

          function factorial(num){
            if(num === 0){
              return 1;
            }
            //define a resul variable
            var result = num;
            //calculate factorial and store value in result
            for(var i = num-1; i >= 1; i--){
              result *= i;
            }
            //return the result variable
            return result;
          };

And we have one more way to do it with recursiveness, just by using and if statement

          function factorial(num){
            if(num === 0){
              return 1
            }
            return num * factorial(num - 1);
          };

3 - Write a function kebabToSnake() which takes a single kebab-cased string and returns the snake_case version.

Code example:

          function kebabToSnake(str){
            var kebab = str;
            var snake = kebab.replace(/-/g,'_');
            return snake;
          };
