Array problem set

Do the following:

  • Write a function printReverse() that takes an a array as an argument and prints out the elements in the array in reverse order (don't actually reverse the array itself) 
                  printReverse([1,2,3,4]);
              //4
              //3
              //2
              //2
              //1

              printReverse(['a','b','c']);
              //c
              //b
              //a
  • Write a function isUniform() that takes an a array as an argument and returns true if all elements in the array are identical. 
                  isUniform([1,1,1,1]);     //true
              isUniform([2,1,1,1]);     //false
              isUniform(['a','b','p']); //false
              isUniform(['b','b','b']); //true
    Hint: use a loop.
  • Write a function sumArray() that accepts an array of numbers and returns the sum of all numbers in the array. 
                  sumArray([1,2,3]);     //6
              sumArray([10,3,10,4]); //27
              sumArray([-5,100]);    //95
  • Write a function maxNumber() that accepts an array of numbers and returns the maximum number in the array. 
                  maxNumber([1,2,3]);     //3
              maxNumber([10,3,10,4]); //10
              maxNumber([-5,100]);    //100

Solution:

  • function printReverse() 
                  var myArray = [];

              function printReverse(myArray){
                myArray.reverse();
                myArray.forEach(function(item){
                  console.log(item);
                });
              }

              printReverse(['a','b','c'])
              // c
              // b
              // a
  • function isUniform() 
                  var myArray = [];

              function isUniform(myArray) {
                for(var i = 0; i < myArray.length - 1; i++) {
                  if(myArray[i] !== myArray[i+1]) {
                    return false;
                  }
                }
                return true;
              }
              isUniform([1,1,1,2])
              //false
              isUniform([1,1,1,1])
              //true
  • function sumArray() 
                  function sumArray(myArray){
                for(var i = 0; i < myArray.length - 1; i++) {
                  var sum = i + (i+1);
                }
                console.log(sum);
              }
              sumArray([1,1,1,2])
              //2
  • function maxNumber() 
                  function maxNumber( MyArray ){
                return Math.max.apply( null, MyArray );
              };

              maxNumber([-5,100]);
              //100
