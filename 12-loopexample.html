Loop example

In this example some name or "String" must be typed in the prompt field and it will be printed in the console. The while loop is used in order to print each letter of the string in the following function:

          // string we're looping over
          var str = "hello",
          //first character is at index 0
              count = 0;

          while(count < str.length){
            console.log(str[count]);
            count++;
          }

Anothe example

Even though the exercise was about a console.log message, the result can be shown in the HTML as well just by adding a few lines of code.

          var str = prompt("Enter your name"),
              count = 0,
              userResults = document.getElementById("user-results"),
              message;

          while(count < str.length){
            var textNode = document.createElement("li");
            message = document.createTextNode(str[count]);
            console.log(str[count]);
            textNode.appendChild(message);
            userResults.appendChild(textNode);
            count++;
          }

Infinite Loops

An infinite loop occurs when the terminating condition in a loop is never true

          var count = 0;

          while (count < 10) {
            console.log(count);
          }

The above example prints "0" over and over because count is never incremented.

Never do this! Unless you want your browser to crash.
