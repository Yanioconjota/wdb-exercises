JavaScript Selectors Exercise

Come up with 4 different ways to select the firs < li > tag

Example:

<h1>I am a dickhead!</h1> <ul class="list-unstyled"> <li id="first" class="special">Hello! I'm so fucking special</li> <li class="special">Goodbye</li> <li>Hi againg peckerhead!</li> <li id="last">Goodbye assholes!</li> </ul>

Result

          var tag = document.getElementById("first");
          console.log(tag);

          var tag = document.getElementsByTagName("li");
          console.log(tag[0]);

          var tag = document.getElementsByClassName("special");
          console.log(tag[0]);

          var tag = document.querySelector(".special");
          console.log(tag);
